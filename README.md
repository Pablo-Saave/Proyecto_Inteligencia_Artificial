# Evaluación 1 - Preparación de Datos

## Contexto
La empresa ficticia **StreamFlow** ofrece contenido multimedia bajo suscripción (Basic, Premium, Enterprise).  
Durante el último año se ha observado un aumento en la tasa de abandono de clientes (**churn**), afectando los ingresos recurrentes.  
El objetivo del proyecto es **preparar los datos** para desarrollar un modelo predictivo que identifique clientes con mayor riesgo de abandono.
